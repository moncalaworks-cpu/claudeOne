# Interactive Workshop Template

**Related Requirement:** REQ-008

Template for running workshops teaching agent development.

---

## Workshop: Building Your First Agent (4 hours)

### Learning Outcomes
- Understand agentic workflows
- Build a functional agent using Claude Code
- Deploy and monitor an agent
- Troubleshoot common issues

### Prerequisites
- GitHub account
- Claude Code CLI installed
- Basic Python knowledge (helpful)
- Anthropic API key (or use Claude Code CLI)

### Agenda

| Time | Topic | Duration |
|------|-------|----------|
| 9:00-9:15 | Welcome & Overview | 15 min |
| 9:15-9:45 | Architecture & Concepts | 30 min |
| 9:45-10:30 | Hands-On: Simple Agent | 45 min |
| 10:30-10:45 | Break | 15 min |
| 10:45-11:30 | Hands-On: GitHub Integration | 45 min |
| 11:30-12:00 | Deployment & Monitoring | 30 min |

### Materials Provided
- Presentation slides
- Code templates
- Example projects
- Troubleshooting guide

### Lab Exercises

#### Lab 1: Simple Agent (45 min)
- [ ] Set up environment
- [ ] Create first agent
- [ ] Implement logging
- [ ] Add error handling
- [ ] Test locally

#### Lab 2: GitHub Integration (45 min)
- [ ] Read GitHub issues
- [ ] Process with Claude
- [ ] Create pull requests
- [ ] Update project boards
- [ ] Deploy to main

## Facilitator Guide

### Setup (Day Before)
- [ ] Test all examples run correctly
- [ ] Prepare slides
- [ ] Set up sample repositories
- [ ] Verify internet connectivity
- [ ] Have backup internet hotspot

### During Workshop
- [ ] Start on time
- [ ] Take attendance
- [ ] Show screen with large font
- [ ] Go slowly - pause for questions
- [ ] Have someone help participants
- [ ] Use breakout rooms if online

### After Workshop
- [ ] Collect feedback
- [ ] Send follow-up resources
- [ ] Answer post-workshop questions
- [ ] Record for later viewing

---

## Troubleshooting Common Issues

| Issue | Solution |
|-------|----------|
| API key not working | Use Claude Code CLI instead (no API) |
| Git command fails | Check git is installed: `git --version` |
| Agent crashes | Check logs for error messages |
| Network timeout | Use retry logic, check internet |
